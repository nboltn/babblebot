youtube
-------

return the url to the latest youtube video:

    ``(youtube:latest-url [id])``

return the title of the latest youtube video:

    ``(youtube:latest-url [id])``

.. rubric:: Notes

.. code-block:: text

    The id for https://www.youtube.com/watch?v=qqC-NjCayAA would be "qqC-NjCayAA".
